*Значение, представляющее метод оплаты*

| Идентификатор | Описание                                                     |
| ------------- | ------------------------------------------------------------ |
| type          | Тип метода (`CreditCard`, `BankTransfer`, `PayPal`).         |
| details       | Данные метода оплаты (номер карты, банковский счет и т. д.). |
